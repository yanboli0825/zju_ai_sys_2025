# Introduction

本项目基于文心LLM API构建了一个中医辨证系统。



项目结构说明：

`datasets`文件夹里存放了MO平台给的原始训练数据，这里我作为喂给大模型的语料。

`predict.py`文件里存放了接口函数及其调用函数。你可以直接在MO平台的jupyter中新建一个cell，将`predict.py`文件内的代码直接复制到新建的cell中进行测试。

> 请注意，由于`temperature`和`top_p`参数的设置不同，大模型的返回结果具备一定的多样性，且MO平台的测试数据可能随机的（至少提交测试处写的是随机），因此多次测试的结果可能不同。
>
> 在我测试的过程中，测试分数可能会落在 [80, 95] 这个区间内，如果你对分数有追求，可以多次提交测试，直到获得满意的测试分数。每次测试的时间大概在几十秒。
