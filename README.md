# Introduction

《人工智能算法与系统》是浙江大学电子信息类专业型研究生的必修课。
本项目为浙江大学2025学年秋学期《人工智能算法与系统》课程实训的参考资料，课程实训的三个主要项目为：

+ 金融异常预测
+ 手写数字识别与垃圾分类
+ 中医辨证系统实现

课程使用MO平台进行作业测试与提交。模型训练既可以在MO平台上使用分配的GPU展开（存在使用时间限制），也可以将所需文件压缩后下载下来，在自己可用的平台上进行训练。
本项目给出了三项课程实训的参考资料，供学习使用。其中，金融异常预测由 @dakeya666 完成，手写数字识别与垃圾分类和中医辨证系统实现由 @yanboli0825 完成。为了精简项目，隐去了部分作业的模型训练过程相关的文件。
